\doxysubsection{Detailed Description}
Collection of named attribute containers. 

One container can be selected as the current container. Most operations are forwarded to the current container, allowing UI code to interact with a single \doxylink{classmeta_1_1AttributeContainer}{Attribute\+Container} interface while the underlying settings group changes at runtime.
