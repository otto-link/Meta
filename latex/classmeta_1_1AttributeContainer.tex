\doxysubsection{Detailed Description}
Container of named runtime attributes. 

Provides\+:
\begin{DoxyItemize}
\item ownership of attributes
\item fast lookup by name
\item type-\/safe access
\item iteration support
\item JSON serialization/deserialization 
\end{DoxyItemize}
