\doxysubsection{Detailed Description}
Base class for all attribute types stored in the reflection system. 

Provides a uniform interface for\+:
\begin{DoxyItemize}
\item runtime type erasure
\item generic value access
\item string and JSON serialization
\item dynamic modification via std\+::any
\end{DoxyItemize}

All concrete attributes are expected to derive from this class.
