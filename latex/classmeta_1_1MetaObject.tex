\doxysubsection{Detailed Description}
Base class enabling runtime metadata attachment. 

\doxylink{classmeta_1_1MetaObject}{Meta\+Object} provides a unified way to attach arbitrary key/value metadata to any object in the Meta framework.

Typical uses include\+:
\begin{DoxyItemize}
\item GUI reflection hints (slider ranges, widgets, grouping)
\item Serialization annotations
\item Editor-\/specific configuration
\item User-\/defined extensions
\end{DoxyItemize}

Ownership model\+:
\begin{DoxyItemize}
\item Metadata is fully owned by the object
\item No shared or global state
\item Lifetime tied to the object 
\end{DoxyItemize}
