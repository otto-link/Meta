\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T$>$\newline
class meta\+::\+Attribute$<$ T $>$}
Typed runtime attribute with reflection support. 

Command that sets an \doxylink{classmeta_1_1Attribute}{Attribute} value with undo/redo support.

Stores a strongly typed value with runtime introspection, conversion, and optional metadata.


\begin{DoxyTemplParams}{Template Parameters}
{\em T} & Stored value type.\\
\hline
\end{DoxyTemplParams}
Stores both previous and new values and supports optional merging for continuous edits (e.\+g. sliders or drag operations).
